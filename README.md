This project is created as part of an internship task to demonstrate how AI can be used for marketing content creation.

📌 Task 2: UGC Ads Generator

This module focuses on creating engaging User Generated Content (UGC) style ads for social media platforms like Instagram Reels.

🔹 Features:
Scroll-stopping hooks (first 3 seconds)
Relatable problem statements
Product-based solutions
Short video ad scripts
Strong call-to-action (CTA)
🔹 Example Use Case:

Product: Face Wash
The system generates hooks, scripts, and ad content that can be directly used for reels or short videos.

🎯 Goal:

To create natural, engaging, and high-converting ad content using AI.
